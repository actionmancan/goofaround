**Week Before:**
- Sign up for deal alerts
- Create accounts on sites you'll buy from
- Add items to wishlists/carts

**Thanksgiving Day:**
- Many deals start evening of Thanksgiving
- Check Amazon, Best Buy, Newegg

**Black Friday:**
- Best deals usually drop
- Monitor Lightning Deals on Amazon

**Cyber Monday:**
- Often better for electronics
- Last chance for many deals

## Purchase Priority

If budget is limited, buy in this order:

1. **Server** - Core of everything
2. **Zigbee coordinator** - Cheap, essential for smart home
3. **PoE switch** - If using IP cameras
4. **2x Cameras** - Start small, expand later
5. **Storage** - Can add later, but plan ahead
6. **Smart bulbs** - Start with 3-5
7. **More cameras** - Expand coverage
8. **Sensors & plugs** - Add as needed
9. **Coral TPU** - Performance upgrade

## Notes

- **Don't overspend on cameras initially**: Start with 2-4, ensure system works, then expand
- **RAM is cheap**: 32GB is sweet spot for home server
- **Storage**: Start with 4TB, add more as needed
- **Zigbee > Wi-Fi**: For smart home devices, Zigbee is more reliable
- **Used is okay**: Enterprise servers are built to last
- **Check return policies**: Essential for Black Friday purchases
